11.45 示例


11 -> 1011
```
11 ÷ 2 = 5  -> 1
5  ÷ 2 = 2  -> 1
2  ÷ 2 = 1  -> 0
1  ÷ 2 = 0  -> 1
```

45 -> 0.011100110011... 循环
```
0.45 × 2 = 0.9     → 0
0.9  × 2 = 1.8     → 1
0.8  × 2 = 1.6     → 1
0.6  × 2 = 1.2     → 1
0.2  × 2 = 0.4     → 0
0.4  × 2 = 0.8     → 0
0.8  × 2 = 1.6     → 1
0.6  × 2 = 1.2     → 1
...
```
